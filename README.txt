Bonjour,

pour faire fonctionner le programme il vous faut installer les librairies python matplotlib, pandas, numpy, csv sur votre machine.

Une fois cela réalisé vous pouvez télécharger puis lancer le programme "CovidOnline.py" avec python depuis votre ordinateur.
Le programme peut mettre un peu de temps à s'exécuter car celui-ci télécharge les données directement sur le site du gouvernement. 
Attention les émulateurs python en ligne ne pourront supporter le fichier des données récupéré sur le site du gouvernement.

Le graphe de l'évolution du nombre d'hospitalisés de 03/2020 à 04/2022 est disponible dans ce projet.
