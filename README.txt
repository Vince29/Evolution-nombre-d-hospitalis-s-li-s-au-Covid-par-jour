Bonjour,

pour faire fonctionner le programme il vous faut installer les librairies python matplotlib, pandas, numpy, csv.

Une fois cela réalisé vous pouvez lancer le programme "CovidOnline.py" avec python depuis votre ordinateur.
Le programme peut mettre un peu de temps à s'exécuter car celui-ci télécharge les données directement sur le site du gouvernement. 
Attention les émulateurs python en ligne ne pourront supporter le fichier des données récupéré sur le site du gouvernement.
